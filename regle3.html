Le Master Mind !

Le Master Mind est un projet développé en langage JAVA, lors du 2ème semestre en L2 EEEA. C'est un jeu qui se joue uniquement dans le terminal et qui se joue à deux.

    • le premier joueur (appelé ≪ codemaker ≫) choisit un ≪ code ≫ composé de 4 couleurs choisies parmi 6, qu’il place dans l’ordre de son choix dans une partie cachée du plateau de jeu (en bas de l’image ci-contre)
  • • le second joueur (appelé ≪ codebreaker ≫) doit retrouver ce code :
  • ⋄ il place sur une première rangée (en haut de l’image) une suite de couleurs ; le codemaker lui indique combien de couleurs sont bien placées (petits pions rouges, sur la droite), et combien de couleurs sont mal placées (petits pions blancs, sur la droite)
  • ⋄ s’il a correctement placé toutes les couleurs, le codebreaker a gagné
  • ⋄ sinon, il recommence sur la rangée suivante
  • ⋄ s’il remplit toutes les rangées du plateau sans avoir trouvé le bon code, il a perdu

Dans la variante qu'on a :

  • il y a 6 couleurs possibles, que l’on représentera par les entiers 1, 2, 3, 4, 5, 6
  • le code est composé de 4 couleurs toutes différentes (pas de doublon) et sans trou laissé sans couleur
  • le codebreaker dispose de 8 rangées pour trouver le code
  • le codebreaker n’a pas le droit de proposer un code avec 2 fois la même couleur (ce n’est pas le cas sur l’image) ou un trou
